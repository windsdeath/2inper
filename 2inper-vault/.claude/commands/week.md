---

이번 주(월~일) 기준으로 vault 현황을 빠르게 요약합니다. **읽기 전용**입니다.

## 절차

1. 오늘 날짜 기준 이번 주 범위 계산 (월요일 ~ 일요일)
2. **이번 주 미팅 노트** — `raw/meetings/YYYY/MM/`에서 이번 주 날짜 파일 목록
   - 각 파일의 `attendees`, `decisions`, `action_items` 추출
3. **진행 중 프로젝트** — `raw/projects/`의 모든 index.md에서:
   - `phase`가 `brief` / `in-progress` / `review`인 것
   - `due_date`가 2주 이내인 것 → 강조
4. **미완료 액션아이템** — 미팅 노트에서 `- [ ]` 체크박스 미체크 항목 추출
5. **이번 주 신규 클라이언트** — `raw/clients/`의 `created` 날짜가 이번 주인 것
6. 결과를 다음 포맷으로 콘솔 출력:

```
## 이번 주 현황 (YYYY-MM-DD ~ YYYY-MM-DD)

### 미팅
- [날짜] 미팅명 — 핵심 결정사항 1줄

### 진행 중 프로젝트
- 프로젝트명 | phase | due: 날짜 ⚠️(2주 이내면 경고)

### 미완료 액션아이템
- [ ] 아이템 (출처: 미팅명)

### 신규 클라이언트
- 클라이언트명
```

7. **저장하지 않습니다.** 요약 보고용 읽기 전용 커맨드.
   - 저장이 필요하면 `/ask 이번 주 현황 정리해줘`를 쓰세요.
