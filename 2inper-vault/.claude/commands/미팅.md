---

회의 노트를 다음 절차로 정리합니다:

1. 오늘 날짜 확인 (`date +%Y-%m-%d`)
2. `templates/meeting-template.md`를 `raw/meetings/YYYY/MM/YYYY-MM-DD-$ARGUMENTS.md`로 복사
3. `YYYY/MM/` 폴더가 없으면 생성
4. frontmatter:
   - `attendees:` 배열에 참석자 wikilink (사람·클라이언트 entity)
   - `client:` 클라이언트 entity wikilink
   - `project:` 프로젝트 entity wikilink (있으면)
5. 안건·결정사항·액션아이템을 채움
6. **녹음 파일이 있으면** 같은 이름 + `.m4a` 또는 `.mp3`로 옆에 두라고 사용자에게 안내. Graphify가 다음 watch 사이클에 자동 전사함
7. 오프-더-레코드 발언은 절대 노트에 옮기지 말 것
8. git 커밋: `raw/meetings: <회의명>`

내가 줄 정보: $ARGUMENTS
