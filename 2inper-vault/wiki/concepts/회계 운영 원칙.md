# 회계 운영 원칙

2INPER의 회계는 **간단·일관·세무사와의 협업** 세 원칙을 따릅니다.

## 사업자 형태

- 형태: (간이과세자 / 일반과세자 / 법인) — 두 분이 결정해 채워주세요
- 사업자등록번호: 별도 보안 저장
- 세무사: (있다면 연락처) — 별도 보안 저장

## 입출금 분리

- 사업용 통장 1개
- 개인용 통장과 절대 혼용 금지
- 카드는 사업용 1장으로 통일

## 매출 처리

- 견적 → 계약 → 선금 청구 → 작업 → 잔금 청구 → 세금계산서 발행
- 모든 매출은 [[raw/finance/README|raw/finance/invoices/]]에 기록

## 매입 처리

- 적격증빙 (세금계산서·신용카드매출전표·현금영수증) 외에는 비용 처리 어려움
- 모든 매입은 [[raw/finance/README|raw/finance/expenses/]]에 기록

## 정기 작업

| 주기 | 작업 | 담당 |
|------|------|------|
| 매주 금요일 | 주간 매출/매입 정리 | [[skills/accountant/SKILL\|경리 Agent]] |
| 매월 1일 | 월간 정산 리포트 | [[skills/accountant/SKILL\|경리 Agent]] |
| 분기별 | 부가세 신고 자료 준비 | [[skills/accountant/SKILL]] + 사람 + 세무사 |
| 연 1회 | 종합소득세/법인세 | 사람 + 세무사 |
