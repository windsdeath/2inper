# 웹 제작 워크플로우

표준 웹사이트 제작 프로세스. 프로젝트마다 약간 변형되지만 골격은 동일.

## 단계

1. **브리프 (planner)** — 클라이언트 인터뷰, 요구사항 명세서, 사이트맵, 와이어프레임
2. **디자인 (designer + character)** — 무드보드 → 핵심 페이지 디자인 → 디자인 시스템
3. **개발 (developer)** — 프론트엔드 → 백엔드/CMS → 통합
4. **QA (qa)** — 디자인 가이드 준수, 크로스브라우저, 반응형, 접근성, 성능
5. **런칭 (developer + planner)** — 배포, DNS, 기본 SEO 셋업
6. **유지보수** — 월정액 또는 단발 패치

## 게이트 (사람 승인 필요)

- 와이어프레임 확정 후 → 디자인 진입
- 디자인 시안 확정 후 → 개발 진입
- QA 통과 후 → 런칭

## 표준 산출물

- 사이트맵 / 와이어프레임 / 디자인 시안 / 코드 / QA 리포트 / 런칭 체크리스트 / 인수인계 문서
