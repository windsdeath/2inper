# 품질 체크리스트

QA Agent가 모든 산출물에 대해 점검하는 체크리스트. Haiku 모델로도 처리 가능한 정형 작업.

## 디자인 산출물

- [ ] [[wiki/concepts/디자인 시스템]] 컬러 토큰만 사용했는가
- [ ] 폰트가 라이선스 범위 내인가
- [ ] 캐릭터의 [[wiki/concepts/캐릭터 IP 가이드|IP 사용 범위]]를 벗어나지 않는가
- [ ] 모바일 폭에서 가독성이 유지되는가

## 카피

- [ ] [[wiki/concepts/브랜드 보이스]] 톤 일치
- [ ] 오탈자 없음
- [ ] 금지 표현 없음
- [ ] CTA가 명확한가

## 코드 / 웹사이트

- [ ] 크롬·사파리·웨일에서 정상
- [ ] iOS·안드로이드 모바일 정상
- [ ] Lighthouse 점수 (Performance ≥ 80, Accessibility ≥ 90)
- [ ] alt 텍스트가 모든 이미지에 있는가
- [ ] 메타 OG 태그 완비
- [ ] favicon 설정
- [ ] 404 페이지 존재

## 법적

- [ ] 개인정보 처리방침 링크
- [ ] 이용약관 (필요 시)
- [ ] 사업자 정보 표기
