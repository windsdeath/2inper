# templates/

새 entity를 만들 때 복사해서 쓰는 표준 템플릿. Obsidian의 Templates 코어 플러그인에 이 폴더를 등록해두면 단축키로 삽입할 수 있습니다.

## 템플릿 목록

- [[templates/client-template]] — 새 클라이언트 entity
- [[templates/project-template]] — 새 프로젝트 entity
- [[templates/character-template]] — 새 캐릭터 entity
- [[templates/license-template]] — 라이선스/계약 entity
- [[templates/meeting-template]] — 회의 노트
- [[templates/proposal-template]] — 제안서 골격
- [[templates/query-template]] — 질의 응답 페이지
- [[templates/concept-template]] — 새 개념 시드 페이지

## 사용 규칙

1. 템플릿을 복사할 때 frontmatter의 `created`를 오늘 날짜로 갱신
2. wikilink가 깨지지 않도록 `[[...]]` 안의 이름을 정확히
3. 의심되면 [[skills/knowledge-manager/SKILL|Knowledge Manager Agent]]에게 검수 요청
