# raw/brand-assets/

**2INPER 자체의** 브랜드 자산. 클라이언트 자산은 [[raw/clients/README|clients/]]에 들어갑니다.

## 들어가야 하는 것

- 로고 파일 (다양한 변형)
- 컬러 팔레트 정의
- 타이포그래피 가이드
- 톤앤매너 레퍼런스
- 회사 소개서·포트폴리오 원본
- 명함·이메일 서명·SNS 프로필 이미지

## 권장 구조

```
brand-assets/
├── logo/
├── colors.md
├── typography.md
├── voice/                        ← 브랜드 보이스 레퍼런스
├── deck/                         ← 회사 소개 키노트
└── social/                       ← SNS 프로필·커버 이미지
```

핵심 컨셉은 [[wiki/concepts/브랜드 보이스]]와 [[wiki/concepts/디자인 시스템]]에 컴파일됩니다.
