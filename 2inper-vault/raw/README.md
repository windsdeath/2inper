# raw/

**입력 전용 영역.** 이 폴더 아래 모든 파일은 사람과 에이전트가 자유롭게 추가/수정할 수 있지만, **LLM Wiki 컴파일러와 Graphify는 절대 이 폴더를 수정하지 않습니다.** 이게 Karpathy LLM Wiki 패턴의 핵심입니다.
